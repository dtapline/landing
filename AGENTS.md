# Agent Guidelines

VitePress documentation/landing page for Dtapline.

## Quick Reference

- **Package manager:** pnpm (v9.10.0)
- **Build:** `pnpm build`
- **Dev:** `pnpm dev`
- **Tests/Linting:** None configured - verify changes manually

## Brand Consistency

Always use **OKLCH color space** and **0.625rem border radius** across all components.
